Variables

  • let. Las variables pueden ser escritos en otras partes del codigo.
  • const. Se utiliza para almacenar valores que no van a cambiar durante todo el codigo.

Que es un algoritmo?

Es un conjunto de instrucciones bien definidas, ordenadas y finitas que que permiten realizar una tarea o alcanzar un objetivo. El algoritmo cuenta con 3 partes:

  • Datos de Entrada o Estado Inicial
  • Proceso
  • Salida o Estado Final


Operadores

  • Aritmeticos. Calcular operaciones matematicas. +, -, *, /
  • Comparacion. Comparaciones entre variables. mayor, menor, igual(===), distinto(!=), menor/igual, mayor/igual.
  • Logicos. Conclusiones logicas entre variables. y(&&), o(||), no(!).


Condicionales

if / else

Ayudara a tomar decisiones en el flujo de nuestros programas. Provee una forma de ejecutar una porcion de codigo, segun una condicion logica.
